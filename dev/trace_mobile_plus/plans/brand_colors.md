# Trace+ Mobile App - Brand Color Scheme

## Overview
Color palette defined based on the Trace+ logo (calculator with coins).

## Primary Colors

### Primary - Dark Charcoal
- **Hex**: `#2D3436`
- **Usage**: App bars, headers, primary backgrounds
- **Description**: Professional dark gray from calculator body

### Primary Dark
- **Hex**: `#1E272E`
- **Usage**: Status bar, darker UI elements
- **Description**: Deeper charcoal for contrast

## Accent Colors

### Accent - Royal Blue
- **Hex**: `#5B7EC4`
- **Usage**: Buttons, links, highlights, active states
- **Description**: Blue accent from calculator CE/C button
- **Contrast**: Accessible on white and dark backgrounds

### Accent Light
- **Hex**: `#7B9DD9`
- **Usage**: Hover states, lighter accents
- **Description**: Lighter variation for subtle highlights

## Neutral Colors

### Background Light
- **Hex**: `#F5F6FA`
- **Usage**: Screen backgrounds, card backgrounds
- **Description**: Soft off-white for content areas

### Surface
- **Hex**: `#FFFFFF`
- **Usage**: Cards, dialogs, elevated surfaces
- **Description**: Pure white for primary surfaces

### Text Primary
- **Hex**: `#2D3436`
- **Usage**: Primary text, headings
- **Description**: Dark charcoal for maximum readability

### Text Secondary
- **Hex**: `#636E72`
- **Usage**: Secondary text, captions, hints
- **Description**: Medium gray for supporting text

## Supporting Colors

### Success Green
- **Hex**: `#00B894`
- **Usage**: Success messages, positive indicators
- **Description**: Professional teal-green

### Error Red
- **Hex**: `#D63031`
- **Usage**: Error messages, validation errors
- **Description**: Clear error red with good contrast

### Warning Orange
- **Hex**: `#FDCB6E`
- **Usage**: Warnings, pending states
- **Description**: Warm orange for attention

### Info Blue
- **Hex**: `#0984E3`
- **Usage**: Information messages, tips
- **Description**: Bright blue for informational content

## Gradient Options

### Primary Gradient
- **Colors**: `#2D3436` → `#1E272E`
- **Usage**: Splash screen background, app bar gradients
- **Direction**: Top to bottom

### Accent Gradient
- **Colors**: `#5B7EC4` → `#7B9DD9`
- **Usage**: Buttons, CTAs, highlights
- **Direction**: Left to right or top to bottom

## Silver/Metallic (from coins)

### Silver Light
- **Hex**: `#DFE6E9`
- **Usage**: Borders, dividers, subtle backgrounds
- **Description**: Light silver from coin highlights

### Silver Medium
- **Hex**: `#B2BEC3`
- **Usage**: Disabled states, inactive elements
- **Description**: Medium silver for muted elements

### Silver Dark
- **Hex**: `#636E72`
- **Usage**: Icons, secondary elements
- **Description**: Darker silver for contrast

## Accessibility Notes

- All text colors meet WCAG AA standards for contrast ratio (4.5:1 minimum)
- Primary blue accent (#5B7EC4) has 4.6:1 contrast ratio on white
- Dark charcoal (#2D3436) has 12.6:1 contrast ratio on white
- Ensure interactive elements have minimum 48x48dp touch targets

## Usage Guidelines

1. **Backgrounds**: Use `#F5F6FA` for screen backgrounds, `#FFFFFF` for cards
2. **Buttons**: Primary buttons use accent blue, secondary use outlined style
3. **Text**: Use text primary for headings, text secondary for body
4. **Feedback**: Always use appropriate semantic colors (success/error/warning)
5. **Gradients**: Use sparingly for splash screen and key visual elements
